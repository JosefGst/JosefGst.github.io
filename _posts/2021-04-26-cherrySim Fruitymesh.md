# Run

The run file `build\Debug\cherrySim_runner.exe` is the template simulation. Run `build\Debug\cherrySim_tester.exe` and open [localhost](http://localhost:5555/) in the browser to see the simulation.

![](/cherrySim_Fruitymesh/test_simulation.PNG "cherrySim_tester")

# Edit the test script

find `TestClustering.cpp` 

![](/cherrySim_Fruitymesh/testclustering.PNG "Path for TestClustering.cpp")

In `TEST(TestClustering, MyOwnCustomTest)` is our custom simulation.

After editing click `->Build->Build Solution`

![](/cherrySim_Fruitymesh/build.PNG "build")

and rerun the `cherrySim_tester.exe`.
